# Assets Klasörü

Bu klasör projenin statik varlıklarını içerir.

## Yapı

```
assets/
├── images/     # Resimler (PNG, JPG, SVG)
├── icons/      # İkonlar
└── fonts/      # Özel fontlar
```

## Kullanım

```jsx
import logo from '../assets/images/logo.png';
import waterIcon from '../assets/icons/water.svg';
```
